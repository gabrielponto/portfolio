# Portfolio Pessoal
A estrutura do portfólio está definida assim:

- Blocos representam um bloco com texto na página inicial. O bloco pode possuir um fundo com cor ou imagem
- Bloco de Case: Herda o mesmo comportamento do Bloco genérico, porém possui uma relação com um "Case" para abrí-lo com hyperlink
- Case: Cada Case é uma história de desenvolvimento de um projeto Específico
- Skill: As Skills são tecnologias que você domina. Elas são ordenadas por uma nota que vai de 1 a 10. Sendo exibidas primeiro as skills com notas mais altas.
O fundo de cada uma será a cor selecionada e a cor do texto é sempre branca com sombra preta.

Acredito que esse projeto é bastante pessoal. Ele foi desenvolvido baseado em pesquisa, principalmente de portfolios já existentes, mas também no meu gosto pessoal.

Se você achar útil utilizá-lo, faça as alterações necessárias. Atualmente grande parte da apresentação está "hard-coded" no template index.html do módulo *main*
